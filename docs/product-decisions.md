# Product Decisions

This file maps major design choices to evidence sources. The in-app source page mirrors the same logic in a more compact form.

| Decision | Why it exists | Sources |
| --- | --- | --- |
| Separate pre-trip, general, topic, scenario, and rescue modes | Avalanche learning has different use cases: just-in-time route prep, broad refresh, targeted lookup, decision practice, and rescue upkeep. | [Daily Process](https://avysavvy.avalanche.ca/en-ca/the-daily-process), [Avalanche Canada Training](https://avalanche.ca/training), [AIARE Recreational Program](https://avtraining.org/recprogram/) |
| Use retrieval cards as the default interaction | Practice testing is higher value than passive rereading for durable learning. | [APS summary of Dunlosky et al.](https://www.psychologicalscience.org/publications/journals/pspi/learning-techniques.html/comment-page-1), [Karpicke & Roediger (2008)](https://doi.org/10.1126/science.1152408) |
| Use spaced review instead of static checklists | Learners should be brought back to material based on elapsed time and performance, not only by manual choice. | [Cepeda et al. (2006)](https://pubmed.ncbi.nlm.nih.gov/16719566/), [APS summary of Dunlosky et al.](https://www.psychologicalscience.org/publications/journals/pspi/learning-techniques.html/comment-page-1) |
| Shorten intervals for high-stakes topics | Rescue and terrain mistakes carry outsized consequence, so the scheduler should keep them fresher. This is a product inference built on avalanche-risk context plus spacing evidence. | [Companion Rescue](https://avysavvy.avalanche.ca/en-ca/companion-rescue), [ATES](https://avysavvy.avalanche.ca/en-ca/the-avalanche-terrain-exposure-scale), [Cepeda et al. (2006)](https://pubmed.ncbi.nlm.nih.gov/16719566/) |
| Require criterion-level performance for mastery | Opening or reading a topic once is not enough; learners should demonstrate strong recall and application. | [Bloom, Learning for Mastery](https://files.eric.ed.gov/fulltext/ED053419.pdf), [AIARE 1](https://avtraining.org/aiare-level-1/), [Avalanche Canada Training](https://avalanche.ca/training) |
| Put terrain and route choice near the center of the product | Terrain is the stable lever users can control before exposure occurs. | [ATES](https://avysavvy.avalanche.ca/en-ca/the-avalanche-terrain-exposure-scale), [Avaluator Trip Planner](https://avysavvy.avalanche.ca/en-ca/the-avaluator-trip-planner), [Online Trip Planner](https://avysavvy.avalanche.ca/en-ca/online-trip-planner) |
| Teach forecast use through interpretation drills | Learners need to convert danger and problem information into travel consequences. | [Danger Scale](https://avalanche.org/avalanche-encyclopedia/danger-scale/), [Avalanche Problems](https://avalanche.org/avalanche-encyclopedia/avalanche/avalanche-problems/), [Avy Savvy](https://avysavvy.avalanche.ca/en-ca) |
| Keep human factors as a first-class topic | Snowpack knowledge alone does not remove decision traps. | [Human Factors](https://avalanche.org/avalanche-encyclopedia/human/decision-making/human-factors/), [Daily Process](https://avysavvy.avalanche.ca/en-ca/the-daily-process) |
| Include scenario drills, not only flashcards | Avalanche learners must transfer knowledge into route and group decisions. | [Roediger & Butler (2011)](https://pubmed.ncbi.nlm.nih.gov/20951630/), [AIARE 1](https://avtraining.org/aiare-level-1/), [AIARE 2](https://avtraining.org/aiare-level-2/), [Avalanche Canada Training](https://avalanche.ca/training) |
| Maintain a dedicated rescue refresher mode | Rescue is both perishable and high consequence, and the course systems treat it as its own refreshable skill set. | [Companion Rescue](https://avysavvy.avalanche.ca/en-ca/companion-rescue), [Avalanche Canada Training](https://avalanche.ca/training), [AIARE Recreational Program](https://avtraining.org/recprogram/) |
