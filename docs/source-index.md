# Source Index

This repository is intentionally source-auditable. The app content and learning model are grounded in official avalanche-education material and established learning-science literature.

## Avalanche education sources

| Source | Why it is here |
| --- | --- |
| [Avalanche Canada Training](https://avalanche.ca/training) | Defines the AST 1, Companion Rescue Skills, Managing Avalanche Terrain, and AST 2 progression used to shape the app's curriculum scope. |
| [Avy Savvy](https://avysavvy.avalanche.ca/en-ca) | Provides the broad online refresher structure for avalanche basics, terrain, forecasting, planning, and rescue. |
| [ATES](https://avysavvy.avalanche.ca/en-ca/the-avalanche-terrain-exposure-scale) | Grounds the simple / challenging / complex terrain model and terrain classification content. |
| [Avaluator Trip Planner](https://avysavvy.avalanche.ca/en-ca/the-avaluator-trip-planner) | Supports pre-trip route filtering using terrain exposure and daily danger. |
| [Online Trip Planner](https://avysavvy.avalanche.ca/en-ca/online-trip-planner) | Supports mapped, terrain-aware trip planning and just-in-time pre-trip review. |
| [Daily Process](https://avysavvy.avalanche.ca/en-ca/the-daily-process) | Informs the pre-trip and on-trip sequencing used in the review modes. |
| [Companion Rescue](https://avysavvy.avalanche.ca/en-ca/companion-rescue) | Supports the rescue refresher and the app's emphasis on perishable rescue skills. |
| [North American Public Avalanche Danger Scale](https://avalanche.org/avalanche-encyclopedia/danger-scale/) | Supports danger-rating interpretation and travel-advice review. |
| [Avalanche Problems](https://avalanche.org/avalanche-encyclopedia/avalanche/avalanche-problems/) | Supports forecast interpretation through type, distribution, likelihood, and size. |
| [Human Factors](https://avalanche.org/avalanche-encyclopedia/human/decision-making/human-factors/) | Supports the decision-making and heuristic-trap content. |
| [AIARE Recreational Program](https://avtraining.org/recprogram/) | Confirms the U.S. progression of Rescue, AIARE 1, and AIARE 2 and its decision-making emphasis. |
| [AIARE 1](https://avtraining.org/aiare-level-1/) | Supports plan / terrain ID / framework / rescue learning outcomes. |
| [AIARE 2](https://avtraining.org/aiare-level-2/) | Supports advanced terrain, leadership, and more complicated decision-making outcomes. |

## Learning-science sources

| Source | Why it is here |
| --- | --- |
| [Dunlosky et al. summary at APS](https://www.psychologicalscience.org/publications/journals/pspi/learning-techniques.html/comment-page-1) | Supports prioritizing practice testing and distributed practice over rereading-heavy study patterns. |
| [Karpicke & Roediger (2008)](https://doi.org/10.1126/science.1152408) | Supports retrieval-first study loops instead of passive exposure. |
| [Roediger & Butler (2011)](https://pubmed.ncbi.nlm.nih.gov/20951630/) | Supports feedback-backed retrieval and transfer-oriented practice. |
| [Cepeda et al. (2006)](https://pubmed.ncbi.nlm.nih.gov/16719566/) | Supports review scheduling over time instead of massed review. |
| [Bloom, Learning for Mastery](https://files.eric.ed.gov/fulltext/ED053419.pdf) | Supports criterion-based mastery rather than exposure-based completion. |

## Boundaries

- This app is designed to complement formal avalanche education, current local forecasts, and repeated field practice.
- The curriculum structure in this repository is a product synthesis inferred from the official sources above. It is not presented as a substitute for AST, AIARE, or instructor-led rescue practice.
